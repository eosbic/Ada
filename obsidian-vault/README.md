# Obsidian Vault: Segundo Cerebro de ADA

Este vault esta pensado para documentar conocimiento tecnico con enlaces
bidireccionales y explorarlo en Obsidian con Graph View.

## Como abrirlo

1. Instala Obsidian.
2. En Obsidian selecciona `Open folder as vault`.
3. Abre la carpeta `obsidian-vault` de este repositorio.
4. Empieza por la nota `[[00-Inicio]]`.

## Convenciones

- Usa `[[Nombre de Nota]]` para crear enlaces bidireccionales.
- Agrega tags en frontmatter (`tags: [tema1, tema2]`) para filtrar el grafo.
- Crea notas nuevas desde `Templates/Plantilla-Nota.md`.
